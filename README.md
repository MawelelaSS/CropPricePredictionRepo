# Crop Market Prediction

A React + Vite frontend application for crop market prediction with a modular feature-based structure.

## Project structure
- `public/` — static assets
- `src/` — application source code
- `src/components/` — feature components with scoped CSS modules
- `src/data/` — JSON data files for crops, market trends, and predictions
- `src/hooks/` — custom hooks
- `src/assets/` — organized feature assets

## Available scripts
- `npm install` — install dependencies
- `npm run dev` — start development server
- `npm run build` — build for production
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
